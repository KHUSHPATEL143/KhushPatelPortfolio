# Khush Patel — Portfolio

Personal portfolio website showcasing my projects, skills, certifications, and hackathon experiences as a Full-Stack Developer and MERN Stack specialist.

🌐 **Live Website:** [khushpatel.online](https://www.khushpatel.online/)

## Current Version

The currently deployed version is **khush-portfolio** — a modern React-based portfolio built with Create React App.

## Tech Stack

- **React** — Component-based UI
- **React Router** — Client-side routing
- **CSS3** — Custom design system with CSS variables, glassmorphism, and micro-animations
- **Cloudinary** — Image hosting for certificates and hackathon photos

## Features

- 🎨 Dark-themed modern design with amber accent palette
- 👤 Profile photo with animated orbiting tech stack badges
- ⌨️ Typewriter effect cycling through roles
- 🛠️ Skills section with categorized tech stack
- 📂 Project showcase with live demo and GitHub links
- 📜 Certificate gallery with click-to-expand lightbox
- 🏆 Hackathon section with embedded event photos
- 🏅 Achievements section
- 📄 Resume viewer and download
- 📬 Contact form with social links
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🔍 SEO optimized with sitemap and meta tags

## Getting Started

```bash
cd khush-portfolio
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000)

## Deployment

Deployed on **Vercel** with the root directory set to `khush-portfolio`.
